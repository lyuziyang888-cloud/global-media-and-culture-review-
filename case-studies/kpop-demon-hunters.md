# KPop Demon Hunters

## Case Type

Global media / national identity / cultural classification

## What This Case Shows

This case is useful for asking whether a media text should be classified by production company, cast, crew, genre, story, cultural references, location or audience.

## Useful For These Concepts

- global media
- global vs local
- national identity
- Western media
- cultural hybridity
- representation

## Analysis Points

- A media product can include cultural elements from one place while being financed, distributed or produced through companies from another.
- National identity in media is often mixed rather than simple.
- The question "Is this American?" can become an analysis of production, ownership, story and cultural representation.

## Exam Use

Use this case when the question asks whether media nationality is clear, fixed or contested.
